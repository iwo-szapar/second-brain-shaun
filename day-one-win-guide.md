## Prerequisites (15 minutes)

### 1. Choose Your Primary Doctor Client

### 2. Document Their Brand Voice
**File**: `memory/style-voice/doctor-[name].md`

**Fill in these sections:**

## Troubleshooting Your First Attempt

### Problem: Still Took 2+ Hours to Edit
**Likely Cause**: Memory files too sparse
**Solution**:
- Add 3-5 more example posts to brand voice file
- Document specific phrases doctor uses
- Include "voice notes" (how they speak uniquely)
- Add more values/frameworks
- Try again with richer context

### Problem: Tone Doesn't Match
**Likely Cause**: Generic brand template
**Solution**:
- Collect 5-10 of doctor's actual posts
- Analyze voice characteristics in detail
- Add "sounds like / doesn't sound like" examples
- Include personality descriptors
- Test voice-only before full generation

### Problem: Structure Needed Major Rework
**Likely Cause**: Wrong pattern for content type
**Solution**:
- Review the Long COVID example structure
- Choose pattern explicitly in prompt
- Use `/plan` to select structure before generation
- Document which patterns work for which topics

### Problem: Generic Positioning
**Likely Cause**: Positioning file too vague
**Solution**:
- Add 3-5 specific differentiators
- Include unique treatment philosophy
- Emphasize what competitors don't do
- Make positioning explicit in prompt
