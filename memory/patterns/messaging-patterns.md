## Pattern Confidence Levels

- **HIGH**: Worked 5+ times with consistent results
- **MEDIUM**: Worked 2-4 times, shows promise
- **LOW**: Worked once, needs more validation
- **TESTING**: New pattern, being evaluated

## Adding New Patterns

When `/learn` extracts a new pattern:

1. Identify the pattern type (email, value prop, objection, etc.)
2. Document the exact phrase or structure
3. Note the context where it succeeded
4. Start with LOW confidence
5. Track future uses and outcomes
6. Upgrade confidence as validated

### Pattern Template

```markdown
| [Pattern Name] | [Exact phrase or structure] | [When to use] | LOW |
```

### Weekly

- Analyze pattern performance trends
