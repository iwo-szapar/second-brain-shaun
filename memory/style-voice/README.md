## What Goes Here

### For Each Doctor Client
Create a file: `doctor-[name].md`

**Include:**
- Voice characteristics (tone, formality, personality)
- Sounds like / doesn't sound like examples
- 3-5 best post examples
- Key themes and language patterns
- Vocabulary preferences

**Why This Matters:**
Content generation uses these files to match doctor's authentic voice, reducing editing time from hours to minutes.

---

## Quick Start

### Adding Your First Doctor
1. Create `doctor-[name].md`
2. Paste 2-3 of their best social posts
3. Note voice characteristics
4. Test with `/ask "Draft post for Dr. [Name]"`
5. Refine based on results
