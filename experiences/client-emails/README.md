# Client Email Experiences

> History of email drafting projects with learnings

---

## Purpose

Every client email drafted through the system gets saved here with context, outcome, and learnings. This builds episodic memory for future email improvements.

## Usage Pattern

**After Drafting Email**:
1. If email achieved great outcome (fast response, project sold, relationship strengthened)
2. Save to this folder with context
3. After 5+ similar emails, run `/learn`
4. System extracts patterns to `memory/patterns/`
